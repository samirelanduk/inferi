inferi
=======

inferi is a statistics and data science Python library.

Example
-------

  >>> import inferi
  >>> variable = inferi.Variable(11, 45, 23, 12, 10)
  >>> variable.mean()
  20.2
  >>> variable.variance()
  219.7
